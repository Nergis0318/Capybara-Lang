# Capybara-Lang 🐾

**Capybara-Lang**은 Rust로 개발된 실험적인 프로그래밍 언어입니다.  
한국어를 완벽하게 지원하는 고유한 문법 구조를 가지고 있으며, 현재 v0.1.0 초기 개발 단계에 있습니다.

---

## 주요 특징

| 특징 | 설명 |
|------|------|
| 🌏 **한국어 친화적** | 변수명, 주석, 문자열에서 한국어를 완벽하게 지원 |
| ⚡ **Rust 기반** | 성능과 안전성을 위해 Rust로 구현 |
| 🎯 **독창적 문법** | 세미콜론, 괄호, 화살표 블록을 활용한 고유한 구문 |
| 📦 **JSON 지원** | JSON 객체를 기본 데이터 타입으로 내장 지원 |
| 🔀 **유연한 불린 타입** | `true`, `True`, `TRUE`, `TrUe` 등 대소문자 무관 |

---

## 파일 확장자

Capybara-Lang 소스 파일은 `.bara` 확장자를 사용합니다.

```
main.bara
hello_world.bara
프로그램.bara
```

---

## Hello World

가장 간단한 Capybara-Lang 프로그램입니다.

```bara
<print:("안녕하세요, 세계!")>
```

---

## 빠른 시작

### 변수 선언 및 출력

```bara
` 변수를 선언하고 출력하는 예제 `

set;["이름"]:("카피바라")
set;["나이"]:(3)
set;["귀여움"]:(true)

<print:("안녕하세요!")>
<print:("Capybara-Lang에 오신 것을 환영합니다! 🐾")>
```

### 사용자 입력 받기

```bara
` 이름을 입력받아 인사하는 프로그램 `

set;["사용자이름"]:(<input;print:("이름을 입력해주세요: ")>)
<print:("반갑습니다!")>
```

### 조건문 사용하기

```bara
set;["pop"]:(true)

if {<pop> = true}
<-
  <print:("조건이 참입니다! ✅")>
->
el {}
<-
  <print:("조건이 거짓입니다. ❌")>
->
```

## 전체 예제 프로그램

다음은 Capybara-Lang의 여러 기능을 함께 사용하는 예제입니다.

```bara
` Capybara-Lang 인사 프로그램 `

<print:("=== Capybara-Lang 인사 프로그램 ===")>

<print:("정보 요약:")>
<print:("Capybara-Lang에 오신 것을 환영합니다! 🐾")>
```
