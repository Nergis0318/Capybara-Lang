# Capybara-Lang 문법 참조

## 개요

Capybara-Lang은 Rust로 개발된 실험적 프로그래밍 언어로, 한국어 식별자와 문자열을 지원합니다. 프로그램은 `.bara` 파일에 작성됩니다.

## 변수 선언

### 기본 변수 (set)
```
set;["변수명"]:(값)
```

**예시:**
```
set;["이름"]:("카피바라")
set;["나이"]:(5)
set;["활성화"]:(true)
```

### 타입 지정 변수 (var)
```
var;["변수명"]:<타입>;<("값")>
```

**예시:**
```
var;["메시지"]:<str>;<("안녕하세요")>
```

## 데이터 타입

| 타입 | 구문 | 예시 |
|------|------|------|
| 문자열 | `("텍스트")` | `("안녕")` |
| 숫자 | `(숫자)` | `(42)`, `(3.14)` |
| 불린 | `(true)` / `(false)` | `(True)`, `(FALSE)` |
| JSON | `({"key": "value"})` | `({"이름": "홍길동"})` |

**참고:** 불린 타입은 대소문자 구분 없이 동작합니다.

## 입출력

### 출력
```
<print:("출력할 내용")>
<print:(변수명)>
```

### 입력
```
<input;print:("프롬프트 메시지")>
```

### 입력값 사용 (pop)
```
<pop>
```

**예시:**
```
<input;print:("이름을 입력하세요: ")>
set;["입력값"]:(<pop>)
```

## 조건문

### 기본 구조
```
if {조건} <-
  실행문
->
```

### elif (fi)
```
fi {조건} <-
  실행문
->
```

### else (el)
```
el {} <-
  실행문
->
```

**완전한 예시:**
```
if {<pop> = true}
<-
  <print:("참입니다")>
->
fi {<pop> = false}
<-
  <print:("거짓입니다")>
->
el {}
<-
  <print:("잘못된 입력")>
->
```

---

## 반복문

### while문 (wh)
```
wh {조건} <-
  실행문
->
```

**예시 - 1부터 10까지 합계:**
```
set;["합계"]:(0)
set;["i"]:(1)

wh {i <= 10}
<-
  set;["합계"]:(합계 + i)
  set;["i"]:(i + 1)
->

<print:(합계)>
```

## 전체 예시

```
`간단한 프로그램`

set;["이름"]:("카피바라")
set;["나이"]:(3)

<print:("안녕하세요!")>
<print:("제 이름은 카피바라입니다")>

if {나이 >= 3}
<-
  <print:("어른 카피바라입니다")>
->
el {}
<-
  <print:("아기 카피바라입니다")>
->
```

## 문법 규칙 요약

| 기호 | 용도 |
|------|------|
| `;` | 구분자 |

| `<>` | 함수 호출 |
| `{}` | 조건식 |
| `<-` `->` | 블록 경계 |
| `` ` `` | 주석 |
